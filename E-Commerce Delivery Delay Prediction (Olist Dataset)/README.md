# Olist E-Commerce Delivery Delay Prediction

### 📦 Overview
This project analyzes over **100,000 Brazilian e-commerce orders** from the [Olist dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) to uncover the main factors contributing to late deliveries.  
Using **SQL** for data extraction and **Python (pandas, scikit-learn)** for modeling, I built and optimized a **logistic regression classifier** that predicts whether an order will arrive late.

---

### 🎯 Objectives
- Identify the strongest predictors of delayed shipments.  
- Build a model that balances **precision and recall** for real-world decision-making.  
- Visualize the relationship between order, review, and delivery attributes.  
- Provide recommendations to reduce false positives and improve logistics reliability.

---

### ⚙️ Tech Stack
- **Languages:** Python, SQL  
- **Libraries:** Pandas, NumPy, Matplotlib, scikit-learn  
- **Tools:** Jupyter Notebook, MySQL  

---

### 🧠 Key Results
- Model accuracy: **80.8%**  
- Precision improved from **0.117 → 0.148** through threshold tuning.  
- Demonstrated the trade-off between recall and precision for operational reliability.  

---

### 📈 Visuals
Example output plots include:
