# Ad Campaign Performance Analysis

### 📊 Overview
This project evaluates digital ad campaign performance data to determine which factors most strongly influence engagement and conversion rates.  
Using **SQL** for data querying and **Python (pandas, matplotlib, seaborn)** for analysis, I explored how variables like **platform, audience demographics, and budget allocation** affect overall campaign ROI.

---

### 🎯 Objectives
- Identify which ad platforms and audience segments generate the highest engagement (CTR, CPC, conversions).  
- Quantify the relationship between spend, impressions, and conversion efficiency.  
- Visualize trends and comparative performance across multiple campaign dimensions.  
- Provide actionable recommendations to improve targeting and resource allocation.

---

### ⚙️ Tech Stack
- **Languages:** Python, SQL  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn  
- **Tools:** Jupyter Notebook, Google Sheets  

---

### 🧠 Key Results
- Determined key predictors of conversion success, including **platform type** and **ad budget ratio**.  
- Identified underperforming channels leading to **10–15% potential ROI improvement** if reallocated.  
- Built summary dashboards visualizing CTR, CPC, and conversion trends for data-driven decision-making.  

---

### 📈 Visuals
Example output plots include:
- CTR and conversion rate comparisons by platform  
- Spend vs. ROI scatterplots  
- Correlation heatmaps of ad metrics  

---

### 💬 Summary
This project demonstrates how **data analytics and visualization** can directly inform marketing strategy — turning ad performance data into actionable insights that enhance **engagement, ROI, and strategic campaign design**.
